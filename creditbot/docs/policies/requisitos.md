# Requisitos y proceso de CrediBot

## Datos para la precalificación

La persona autoriza la consulta académica simulada y proporciona una cédula ecuatoriana válida, el destino del crédito, monto, plazo e ingreso mensual aproximado.

## Documentos para validación final

La precalificación no reemplaza la revisión humana. Un asesor puede solicitar documento de identidad vigente, comprobante de ingresos y soportes adicionales según el caso.
